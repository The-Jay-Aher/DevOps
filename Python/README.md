# Python

## Basics of Python

### IF - LOOP

```python
cat = "spot"
if "s" in cat:
  print("Found an 's' in a cat")
  if cat == "Sheba":
      print("I found Sheba")
  else:
      print("Some other cat")
else:
  print("A cat without an 's")
```

### FOR - LOOP

```python
for i in range(30):
    print(i)

for i in range(6):
    if i % 2 == 0:
        continue
    print(i)
```
